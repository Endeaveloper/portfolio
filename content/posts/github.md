$ git push -u origin master
To https://github.com/Endeaveloper/portfolio

나의 GIT 원격지로 push를 할려고 명령어를 쳤더니!!! 

 ! [rejected]        master -> master (fetch first)
error: failed to push some refs to 'https://github.com/Endeaveloper/portfolio'
hint: Updates were rejected because the remote contains work that you do
hint: not have locally. This is usually caused by another repository pushing
hint: to the same ref. You may want to first integrate the remote changes
hint: (e.g., 'git pull ...') before pushing again.
hint: See the 'Note about fast-forwards' in 'git push --help' for details.

이런 오류가 나면서 GIT에 push 할 수 없는 상황이 되었다.

push 를 하기 전에
$ git status 를 커맨드에 입력하면,

$ git status
On branch master
Untracked files:
  (use "git add <file>..." to include in what will be committed)

        ../fasdsdf

nothing added to commit but untracked files present (use "git add" to track)

fasdsdf 분양서를 부동산에 가지고 있어야됨 이거를 가지고 팔아버릴수도있고, 할수도있음 이분이 임대인인지 확인해달라 
분양서 주인이 가져가면 안됨 부동산 보관  조합물량이라 등기가 더 오래걸림 입주하고나서 오래걸림
